# AI Spend Audit — Credex Assignment

This repository contains my take-home build for the Credex WebDev assignment: an "AI Spend Audit" web app that helps startups identify overspend on AI tools and recommends cheaper plans or alternatives.

What I'm building
- A public, no-login web app that accepts a user's AI tool list (tool, plan, monthly spend, seats), team size, and primary use case.
- An audit engine that evaluates plan fit and recommends cheaper plans or alternative tools with defensible savings math (sources recorded in `PRICING_DATA.md`).
- A polished results page showing per-tool and total monthly/annual savings, plus sharing via a unique public URL with Open Graph preview.
- A ~100-word personalized summary generated by an LLM (Anthropic preferred) with prompts documented in `PROMPTS.md`.
- Lead capture (email + optional fields) stored in a real backend and sending a transactional confirmation email.

Tech stack (recommended)
- Frontend: Next.js + TypeScript
- Styling: Tailwind CSS (shadcn/ui or similar optional)
- Backend/storage: Supabase (Postgres) for leads and audit records
- LLM: Anthropic (preferred) or OpenAI as fallback
- Email: Resend or Postmark for transactional emails
- Testing: Jest + React Testing Library
- CI/CD: GitHub Actions + Vercel for deployment

Why these choices
- Next.js (Vercel) gives fast SSR/ISR pages for shareable audit URLs and easy OG meta rendering.
- Supabase provides a quick, hosted Postgres and REST API with minimal setup.
- Anthropic matches the brief preference and has free credits for students/startups sometimes; OpenAI is a reliable fallback.

How to run locally

Prerequisites
- Node.js 18+ and npm or pnpm
- A Supabase project (URL and anon key) or an alternative Postgres connection
- API key for Anthropic or OpenAI (for LLM summaries)

Setup

1. Install dependencies

```bash
npm install
```

2. Copy environment variables (example)

Create a `.env.local` with:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
ANTHROPIC_API_KEY=your_anthropic_key
MAIL_API_KEY=your_resend_or_postmark_key
```

3. Run dev server

```bash
npm run dev
```
